CSS

  • Casacading Style Sheets
  • Notes
    • Skeleton:
      selector {
        property: value;
      }
    • color: name, rgb, rgba or #Hexadecimal; // to change selector color
    • background : color or url; // change background color or set to image
    • border: blueviolet solid 3px; // simple way of setting border property
    • Quick Example
    • #idname and id="" // one time identification of element
    • .classname and class"" // multiple elements getting same style
    • * selector for every element on the page
    • Descendent selectors: li a // applies to every anchor tag inside of an li
    • Adjacent selector: h4 + ul // applies to adjacent siblings
    • Atrribute selector: a[href="link"] or input[type="text"]
    • nth of type: ul:nth-of-type(3) // changes every 3rd element of the type
